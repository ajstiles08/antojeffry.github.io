Project information

  • Title: IPL Analysis and Prediction using Machine Learning Algorithms [Best Paper Award NCSET 2022]
  • Category: Python, Data Analysis, Statistical Analysis
  • Project date: 20 September, 2022
  • Project URL: IPL 2022 Prediction

Project Description

  • Employed machine learning models and statistical analysis for IPL 2022 champion prediction.
  • Conducted Exploratory Data Analysis to gather player statistics and analyze all matchups.
  • Considered factors such as venue, toss decisions, run scoring, and team matchups to determine win percentages.
  • Successfully predicted the Gujarat Titans as the IPL 2022 champion.
